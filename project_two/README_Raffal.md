# Project Two: Food Delivery Application

This project is a design for a food delivery application.

## Description

This project designs 6 different screens of a food delivery application using flutter. It supports button control, scrolling, and screen navigation.

## Getting Started

### Executing program

Run the program through main.dart inside project_two/lib

## Program Overview

### Starter screen:
starter.dart support the intial screen the user see when opening the app or after signing out.
the user have two buttons that navigate them to either sign up or sign in screen.

<img width="1440" alt="StarterScreen" src="https://github.com/Raff22/Project-2/assets/86999942/fbea8fb9-6868-43bc-8b66-44ed18dfcf96">

### Sign in screen:
If the user tap on the sign in button in signin.dart this screen will display to the user. The user can then enter their user name and password and hit sign in button to go to home page. If the user press back button or swipe right they will go back to starter screen.
